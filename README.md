# QCM Python

Un questionnaire à choix multiples (QCM) implémenté en Python.

## Fonctionnalités

- 10 questions avec 3 réponses possibles chacune
- Questions et réponses présentées dans un ordre aléatoire
- Interface en ligne de commande simple
- Calcul du score final
- Affichage du corrigé

## Installation
Aucune dépendance externe n'est requise

## Utilisation

Exécutez le programme principal :
```bash
python main.py
```

Pour lancer les tests unitaires :
```bash
python -m unittest test_qcm.py
```

## Structure du projet

- `question.py` : Définition de la classe Question
- `qcm.py` : Logique principale du QCM
- `main.py` : Point d'entrée du programme
- `test_qcm.py` : Tests unitaires
